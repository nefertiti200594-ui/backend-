Spring Boot Book Store API

A REST API built with Spring Boot.

Features

* JWT Authentication
* Spring Security
* PostgreSQL
* Swagger OpenAPI
* CRUD Operations for Books
* DTO Pattern
* Validation
* Global Exception Handling

Technologies

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* PostgreSQL
* Swagger/OpenAPI
* Maven

Authentication Endpoints

* POST /auth/register
* POST /auth/login

Book Endpoints

* GET /books
* GET /books/{id}
* POST /books
* PUT /books/{id}
* DELETE /books/{id}
